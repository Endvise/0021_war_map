⚔️ IR 시뮬레이터 사용법

▶ 빠른 시작

  1. 헤더에서 내 연맹 버튼 선택
  2. 🔨 건설 클릭 → 맵에서 타워 위치 클릭
  3. ▶ 계산 클릭 → 오른쪽 📊 탭에서 결과 확인

🖱 맵 조작

왼쪽 클릭현재 모드로 타워 건설·철거·점화
우클릭(타워)컨텍스트 메뉴: 🔥 점화 · 🔧 철거(반환) · 📍 이동 · 🗑 삭제(반환없음)
우클릭(도시)소유권 직접 지정 메뉴
우클릭(헬기장)연맹 관계 토글 기준점 (재우클릭=해제)
마우스 휠줌인·줌아웃 (좌상단 줌 버튼도 가능)
미니맵 클릭우하단 미니맵 클릭 → 해당 위치로 점프

🗺 경로 모드 (출발·도착·경유지)

🚩 첫 점출발점 — 헬기장/도시/타워 자동 판정 (스냅됨)
🏁 마지막 점도착점
🏴 중간 점경유지 — 순서대로 거쳐갈 위치
우클릭새 점 추가 (자동으로 가까운 앵커에 스냅)
기존 점 우클릭해당 점 제거
점 드래그점 위치 변경 (드래그 중에도 스냅)

🔮 미래 예측 모드

경로 계산 시 만들어질 미래 타워가 점선 박스로 맵에 미리 표시됩니다.

점선 박스아직 건설되지 않은 미래 타워 (예측)
솔리드 박스로 전환UTC 도달 시간이 지나면 자동으로 일반 타워 박스로 전환
호버 툴팁UTC 도달 시간 + 누적 시멘트 비용 + 도시 다리 여부
🏙 표시해당 타워가 도시를 다리로 이용하는 지점
🔨 이 경로대로 건설📊 탭 결과의 버튼 — 시멘트 차감 + 즉시 일괄 건설

🔘 헤더 버튼

🖱 일반맵 탐색 전용 (건설/철거/점화 없음)
🔨 건설클릭 위치에 타워 건설 (시멘트 소모)
🔧 철거타워 클릭으로 즉시 삭제 (시멘트 50% 반환)
🔥 점화타워 클릭 → 내구도 초당 감소 시작
↩ 취소마지막 건설 타워 삭제
