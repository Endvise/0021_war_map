📡 실시간 동기화 (전장코드)

  1. 헤더 하단 전장코드 입력 (예: BATTLE01)
  2. 연결 클릭 → 🟢 연결됨 확인
  3. 연맹원도 동일 코드로 연결 → 맵 수정 시 자동 공유
