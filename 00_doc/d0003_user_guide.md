## 빠른 시작

1. 위 URL 접속
2. **연맹 버튼** 중 내 연맹 선택 (헤더 상단)
3. **🔨 건설** 모드 클릭 → 맵에서 타워 위치 클릭
4. **▶ 계산** 클릭 → 오른쪽 📊 탭에서 결과 확인

---

## 헤더 버튼 설명

### 타워 모드

| 버튼 | 단축키 | 동작 |
|------|--------|------|
| 🖱 일반 | — | 맵 탐색 전용, 건설/철거 없음 |
| 🔨 건설 | — | 클릭 위치에 타워 건설 (시멘트 소모) |
| 🔧 철거 | — | 타워 클릭으로 즉시 삭제 |
| 🔥 점화 | — | 타워 클릭 → 점화 (내구도 초당 감소) |
| ↩ 취소 | — | 마지막 건설한 타워 1개 삭제 |

### 기타 버튼

| 버튼 | 동작 |
|------|------|
| ▶ 계산 | 모든 연맹의 도달 시간·충돌 좌표 계산 |
| 🗑 초기화 | 전체 데이터 삭제 + 새로고침 |
| 🏆 랭킹 | 현재 점수 순위표 팝업 |
| 🗺 경로 | 경유지 깃발 모드 전환 |
| 🤝 동맹 설정 | 연맹 간 동맹 관계 설정 모드 |

---

## 맵 조작

| 동작 | 기능 |
|------|------|
| 왼쪽 클릭 | 타워 건설/철거/점화 (현재 모드에 따라) |
| 오른쪽 클릭 | 경로 모드: 경유지 추가/삭제 / 도시: 소유권 변경 |
| 마우스 휠 | 줌인/줌아웃 |
| 휠 클릭 + 드래그 | 맵 뷰 이동 |
| 타워 드래그 | 타워 위치 이동 |
| 도시 우클릭 | 소유 연맹 변경 |

---

## 경로 모드 (🗺 경로)

1. **🗺 경로** 버튼 클릭
2. 맵에서 **우클릭** → 경유지 깃발 추가
3. 경유지 깃발 **우클릭** → 해당 경유지 삭제
4. 경유지 깃발 **왼쪽 클릭 드래그** → 위치 이동
5. 경유지 변경 시 **📊 탭**에 자동으로 세그먼트별 경로 계산 결과 표시

---

## 탭 패널 (오른쪽)

| 탭 | 내용 |
|----|------|
| 🗺 | 맵 크기, 타워 건설시간, 목표 좌표 설정 |
| ⚙ | 연맹별 시멘트 보유량, 분당 생산량 설정 |
| 🤝 | 연맹 간 동맹·중립·적대 관계 매트릭스 |
| 🏙 | 도시 점령 현황 및 예상 점수 |
| ⏰ | 전장 일정, UTC 시간, 코인추격 쿨타임 |
| 🔥 | 점화된 타워 전소까지 남은 시간 계산 |
| 📊 | 계산 결과: 도달 시간, 충돌 예측, 경로 표 |

---

## 실시간 동기화 (전장코드)

연맹원과 같은 화면을 실시간으로 공유합니다.

1. 헤더 하단 **"전장코드"** 입력란에 코드 입력 (예: `BATTLE01`)
2. **연결** 클릭 → `🟢 연결됨` 표시 확인
3. 연맹원도 동일한 코드로 연결
4. 어느 쪽이든 맵 수정 시 상대 화면에 자동 반영

> 코드는 아무 문자나 가능. 같은 코드를 아는 사람만 접속 가능.

---

## 연맹 설정

1. 오른쪽 ⚙ 탭 → 연맹 이름·색상 편집
2. 헤더 상단 연맹 버튼 클릭으로 현재 활성 연맹 전환
3. 동맹 설정: **🤝 동맹 설정** → 헬기장 클릭으로 선택 → **✓ 완료**

---

## 도시 점령 설정

- 맵에서 도시 아이콘 **우클릭** → 소유 연맹 선택
- 또는 오른쪽 🏙 탭에서 드롭다운으로 선택
- 점령 도시 수에 따라 시멘트 분당 생산량 자동 계산

---

## 저장 / 불러오기

- **자동 저장**: 모든 변경사항이 브라우저 LocalStorage에 자동 저장
- **새로고침 후에도 복원**됨
- **전체 초기화**: 🗑 초기화 버튼 → 확인 클릭
- **JSON 내보내기/불러오기**: 🗺 탭 하단의 저장·불러오기 버튼
